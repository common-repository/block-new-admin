A security plugin to block the creation of new administrators

== Description ==

This plugin prevents the creation of new administrators, either by code, or by acting directly on the database. Ideal for locking possible backdoor hidden in code.
BE CAREFULLY: DEACTIVATING THE PLUGIN YOU DON'T REMOVE THE BLOCK. YOU MUST USE THE PASSWORD IN SETTINGS / BLOCK NEW ADMIN AND DEACTIVATE THE BLOCK BEFORE DEACTIVATING THE PLUGIN

It provides:

1. A setting item to create a password to block o unblock a new administrator creation
	
== Installation ==


1. Upload the plugin files to the `plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Block New Admin settings under Settings to configure the password


== Frequently Asked Questions ==

 = How may I unblock the administrator's creation ? =

Use the Block New Admin settings under Settings, insert the password you chose then click on Deactivate Block
